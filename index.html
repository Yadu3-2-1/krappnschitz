Welcome to My Website

This is a simple example of a mobile-friendly webpage layout. The main content area is centered and has a maximum width, which creates space on both the left and right sides, especially on wider screens like tablets and desktops.

On smaller screens, like a mobile phone, the content will take up most of the screen width but still maintain a small, comfortable margin. This is achieved by setting max-width to a percentage and using margin: auto; in the CSS.
